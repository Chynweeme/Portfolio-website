This is a class selector. Class selectors are styled by first having
the class attribute in the opening tag of the element. Give a name to the
class attribute. Then go to your external css to style the class... To do this,
first add a dot(.) sign before the name of the class in order for the class
selector to work.

Class selector in use

This is ID selector. It is similar to class selector, except that with id selector,
you can only use it for one element at a time. YOu can not use it for multiple elements,
however class selectors can be used for as many elements as you want. We need
id attribute for id selector. After putting the id attribute in the opening tag of the html
file, head over to the external css file and style it using the id selector. To style it in the
external css file, use the ash sign (#) first before the id name.

Will this work?

Child selector is the selector that allows us pick the child of an element. In child
selector, we do not need child attribute... there is nothin like child attrubute.. lol
however, we just need to go to the external css file, pick the name of the parent tag, in
this cas it is "div", give a space and then use the "greater than sign" (>), a space bar
and then the child element, which in this case is the "p" tag.
Note: Always remember to use the greater than sign for the child selector. And never forget
to close your css styling with a semi-colon!
Let me quickly explain descendant selector. A descendant selector is used to pick the child element
of a child element: like the grand child. Lol. To do this, you could either go through the child selector
path and use your greater than sign in two places, or you can just use the descendant selector by
calling out the parent selector and then giving a space and calling the grandchild selector immediately
